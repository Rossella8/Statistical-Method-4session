Chapter 8 - Exercise 1

The following table shows numbers of occasions when inhibition (i.e., no flow of current across a membrane) occurred within \(120\) s, for different concentrations of the protein peptide-C (data are used with the permission of Claudia Haarmann, who obtained these data in the course of her PhD research). The outcome \(\texttt{yes}\) implies that inhibition has occurred.

Use logistic regression to model the probability of inhibition as a function of protein concentration.

Solution

conc <- c(.1, .5, 1, 10, 20, 30, 50, 70, 80, 100, 150)

here’s the conc

no <- c(7, 1, 10, 9, 2, 9, 13, 1, 1, 4, 3)
yes <- c(0, 0, 3, 4, 0, 6, 7, 0, 0, 1 ,7)

here’s no+yes let’s sum

n <- no + yes

plot

plot(log(conc), log((yes+0.5)/(no+0.5)))
